# PlatinumRx Data Analyst Assignment

This project contains SQL, Spreadsheet, and Python solutions demonstrating
data analysis, querying, and automation skills.

## Folder Structure

- **SQL/**: Contains schema setup and solution queries
  - `01_Hotel_Schema_Setup.sql`: Hotel management system tables and sample data
  - `02_Hotel_Queries.sql`: 5 required SQL queries for hotel analysis
  - `03_Clinic_Schema_Setup.sql`: Clinic management system tables and sample data
  - `04_Clinic_Queries.sql`: 5 required SQL queries for clinic analysis

- **Spreadsheets/**: Contains Excel workbook with formulas
  - `Ticket_Analysis.xlsx`: Ticket and feedback analysis with calculated fields

- **Python/**: Contains scripts for time conversion and duplicate removal
  - `01_Time_Converter.py`: Convert minutes to readable hours and minutes format
  - `02_Remove_Duplicates.py`: Remove duplicate characters from strings

## Tools Used

- MySQL / PostgreSQL / SQL Server
- Microsoft Excel / Google Sheets
- Python 3.x

## How to Use

1. **SQL Queries**: Execute schema setup files first, then run the query files to get analysis results
2. **Spreadsheets**: Open the Excel file and view the calculated fields and formulas
3. **Python Scripts**: Run each Python file to see the output of the functions

## Features

- Complete database schemas with relationships
- Complex SQL queries using window functions and CTEs
- Excel formulas for data transformation and analysis
- Python utility functions with type hints
